📝 Todo List App (CRUD with Axios)

A simple and responsive Todo List web application built using HTML, CSS, JavaScript, and Axios.
This project performs full CRUD (Create, Read, Update, Delete) operations using a MockAPI backend.

Live Link: (https://todo-list-sanjai2614.vercel.app/)

🚀 Features

✅ Add new todo items

📖 Fetch and display all todos from API

✏️ Update existing todo items

❌ Delete todo items

🎨 Modern responsive UI with gradient background

🔄 Real-time UI update after every action

🛠️ Technologies Used

HTML5

CSS3

JavaScript (ES6)

Axios (for HTTP requests)

MockAPI (for backend simulation)

The app performs:

GET → Fetch all todos

POST → Add new todo

PUT → Update todo

DELETE → Remove todo
